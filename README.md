Rowing
======

学如逆水行舟，不进则退。

---

## 前言

开源社区中有许多优秀的在线考试系统，它们要么是基于 SSM 框架构建，要么是基于 Spring MVC 框架构建，还有少数部分是基于 Spring Cloud 框架构建。

对个人而言，我只需要一个学习辅助工具，用来检验所学知识的掌握水平，所以它应该尽量精简，不需要有太复杂的功能。
